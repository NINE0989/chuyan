变更内容

  新增库文件（WebEngine/html/static/）

- marked.min.js — Markdown 解析引擎（GFM 支持）
- highlight.min.js — 代码语法高亮（自动语言检测）
- atom-one-dark.min.css — hljs 深色主题（匹配 UI 风格）
- purify.min.js — XSS 防御

  修改文件

  frontend_v2.html — 主要前端

- <head> 新增 atom-one-dark 主题 CSS link
- 库脚本在 inline script 之前加载（marked → hljs → purify → 内联代码 → code-blocks.js）
- 替换 renderMd() 为 marked.parse(text, {breaks:true, gfm:true}) + DOMPurify 过滤
- 新增 highlightBubbleCode() 辅助函数
- 所有渲染路径（addMessage + 6 处 SSE 回调）都调用 highlight
- CSS 新增：h1-h6、strong、em、a、del、th 样式 + hljs 背景兼容

  index.html — 备选前端（同上变更，无 code-blocks.js 依赖）

  code-blocks.js — 纯文本 shader 回退气泡也支持 hljs 高亮

  渲染管线

  文本 → marked.parse() [GFM] → DOMPurify → innerHTML → hljs.highlightElement() → code-blocks.js
